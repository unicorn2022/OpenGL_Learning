# 对VSCode的配置

## 设置编译工具&构建工具

> ctrl + shift + p > cmake: scan for kits

- 扫描工具包

> ctrl + shift + p > cmake: select a kit

- 设置工具包为：**VS x86_amd64**

## 解决输出乱码问题

> ctrl + , > cmake: output log encoding

- 设置输出日志编码为：**utf-8**
